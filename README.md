## Project Overview

Elements@Church is a trendy bar located 111 WF Nkomo Street, at the heart Pretoria's central business district. It's history takes us way back to the 1850's where Church square was origionally Market Square and later hosted 3 successive churches between 1856 and 1905 giving the area it's enduring name. It's main target audience is considered to be young adults and early proffessionals who enjoy night life and socializing.

## Website Goals and Objectives

The main aim is to make the website more interactive and understandable. The current website is more complex and my aim is to make it more simpler to use so that people can take their desired actions with ease. Elements@Church does have a fully functional website but it can be more interactive as the only service they offer from the website so far is booking tables online. The goal is to improve customer engagement, revenue, operational effeciency, and brand presence.
